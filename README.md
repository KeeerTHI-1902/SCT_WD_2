# Stopwatch Web Application ⏱️

## Description
This project is a user-friendly Stopwatch Web Application developed using HTML, CSS, and JavaScript.  
It allows users to start, pause, reset the stopwatch, and record lap times with millisecond precision.

## Features
- Start, Pause, Reset functionality
- Lap time tracking
- Milliseconds support
- Clean and responsive UI
- Runs without backend (client-side only)

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Open the project folder in VS Code
2. Right-click `index.html`
3. Select **Open with Live Server**
